## A 股投研系统说明

当前工作区已经沉淀出一套可复用的 A 股成长股投研工作流，核心能力位于 .github 下：

- agents: 负责分层分析与编排
- prompts: 负责统一输出格式
- skills: 负责工作流与路由规范

系统设计说明见：

- [.github/a-share-growth-system-design.md](.github/a-share-growth-system-design.md)

这套系统当前采用“巴菲特 + 费雪 + 林奇 + 格雷厄姆 + 价值投资 3.0”的四合一标准，核心关注：

- 商业质量与资本配置
- 长期增长跑道
- 护城河是否支撑增长
- 再投资能力
- 管理层质量与资本配置
- 安全边际与失败保护
- 林奇分类、能力圈与故事验证
- Scuttlebutt 式公开信息交叉验证
- 成长溢价是否合理

同时，系统把 MQVGS、市场状态、情绪、组合优化和因子暴露视为可选增强层：只有在公开数据充分时才启用；数据不足时明确降级为定性观察或直接标记为数据不足。
