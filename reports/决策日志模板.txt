二、这笔交易对应的研究前提

本次交易基于此前研究结论：

- 硬阈值层：达标 / 观察 / 淘汰 / 不适用
- 基本面层：优质 / 观察 / 回避
- 估值层：买入 / 观望 / 放弃

核心含义：
- 这笔交易属于什么性质
- 是因为价格进入 Buy Zone，还是因为基本面出现强化
- 这是不是一次试探性动作，而不是最终确认型重仓

三、交易理由

1. 公司质量判断
- 核心商业模式是否成立
- 护城河、ROE、现金流、资产负债表是否支持原判断

2. 价格与赔率判断
- 当前价格相对 Fair Value、Buy Zone、Strong Buy Zone 处于什么位置
- 当前买卖动作的赔率基础是什么

3. 触发这笔交易的具体变化
- 财报验证
- 股价回落
- 市场误杀
- 行业格局改善
- 组合机会成本变化

四、这笔交易不等于什么

1. 不等于已经完全确认原逻辑
2. 不等于风险已经消失
3. 不等于后续可以机械重复同样动作
4. 不等于任何价格都合理

五、后续验证点

1. 下一份年报 / 季报最关键的验证指标是什么
2. normalized FCF 是否继续成立
3. ROE 中枢是否稳定
4. 利润率、周转率、资本回报是否保持趋势
5. 行业竞争、价格战、渠道结构、海外业务或第二曲线是否出现新变化

六、继续加仓条件

1. 基本面继续强化
2. 价格继续回到更有吸引力区间
3. 财报或经营数据验证原假设
4. 与其他候选标的相比，机会成本仍占优

七、暂停加仓条件

1. 股价反弹后安全边际明显收窄
2. 财报未能验证关键假设
3. 现金流、ROE、利润率出现弱化信号
4. 行业竞争或公司执行力出现不利变化

八、失效条件与卖出触发点

1. 原先成立的护城河判断被反证
2. ROE 中枢明显下台阶
3. normalized FCF 持续恶化
4. 资产负债表风险上升
5. 管理层资本配置明显失误
6. 股价显著高于合理价值，而基本面未同步上修

九、当前定位

这笔交易的正确定位是：

- 建仓试单 / 确认型加仓 / 被动减仓 / 主动止盈 / 逻辑失效退出

而不是：

- 情绪化追涨
- 因短期波动做出的临时动作
- 没有研究前提支撑的拍脑袋决策
