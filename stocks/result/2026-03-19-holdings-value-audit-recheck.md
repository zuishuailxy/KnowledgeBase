# 持仓价值投资审判报告复检

- 生成日期：2026-03-19
- 复检方式：利用 A股投研总控台 对原持仓审判报告逐只复核
- 复检对象：原报告中的全部 A 股个股持仓
- 说明：本次复检不是从零重做新报告，而是检查原结论是否站得住，并指出需要修正的地方

## 复检口径

本次复检重点看四件事：

- 长期商业质量是否成立
- ROE 与自由现金流是否支持长期占用资本
- 护城河、定价权、行业结构有没有被原报告说清
- 原报告有没有把仓位问题误写成公司质量问题

边界说明：

- 不做目标价判断
- 银行、资源股不生搬硬套统一 ROE/FCF 硬阈值，而按行业特性单独看
- 重点复检争议票：德业股份、招商银行、中国海油、紫金矿业、分众传媒、顺丰控股

## 总评

### 哪些原结论基本可靠

原结论基本可靠的有两类：

- 保留名单中的高质量资产：贵州茅台、美的集团、五粮液、海大集团、福耀玻璃、恩华药业、比亚迪、中国移动
- 卖掉名单中的明显弱于长期复利标准的资产：亨通光电、南山铝业、海螺水泥

### 哪些原结论需要保守化

- 德业股份：应从观察下调到卖掉
- 分众传媒：应从保留下调到观察
- 顺丰控股：应从卖掉上调到观察
- 招商银行：观察可以维持，但原报告论证过粗
- 中国海油：观察可以维持，但原报告把它说成纯油价映射过于简化
- 紫金矿业：观察可以维持，但原报告对管理层和项目能力的正面因素写得不够

### 哪些仓位问题比个股质量问题更严重

- 德业股份 605117：16.32%，这是明显的仓位错误，且已经超过了普通成长制造应占用的风险预算
- 紫金矿业 601899：6.62%，对于资源股而言，这个仓位已经不是辅助配置，而是实质性的周期暴露
- 招商银行 600036：4.06%，公司质量在银行里不低，但如果组合目标是高确定性长期复利，银行仓位本来就该克制

反过来，优质资产低配同样是问题：

- 五粮液 000858：1.31%
- 恩华药业 002262：2.84%
- 福耀玻璃 600660：2.88%
- 海大集团 002311：3.48%
- 美的集团 000333：3.89%
